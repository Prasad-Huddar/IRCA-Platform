# IRCA Platform - IRCA-Web

## Overview

The IRCA Platform is a comprehensive web application designed to provide accessible information about Integrated Rehabilitation Centre for Addicts (IRCA) centers, hospitals, and psychiatrists across Karnataka, India. This platform serves as a vital resource for individuals seeking addiction recovery services, their families, and healthcare professionals.

## Project Purpose

The platform addresses the critical need for transparent and easily accessible information about rehabilitation services in Karnataka. By centralizing data on government-approved IRCA centers, hospitals, and mental health professionals, it helps bridge the gap between those in need of treatment and available resources.

## Key Features

### Interactive Map
- District-wise visualization of Karnataka
- Clickable markers showing facility locations
- Real-time filtering and search capabilities
- Detailed facility information on marker interaction

### Advanced Search & Filtering
- Search by center name, district, or location
- Filter by facility type (IRCA centers, hospitals, psychiatrists)
- Category filtering (government vs. private)
- Sorting options (name, district, bed capacity)

### Comprehensive Facility Database
- **IRCA Centers**: Government and private rehabilitation centers
- **Hospitals**: Government and private hospitals with psychiatric/de-addiction services
- **Psychiatrists**: Mental health professionals and specialists

### Facility Information
- Detailed center profiles with contact information
- Bed capacity and service offerings
- Verification status for government-approved centers
- Location details and directions

### User Experience Features
- Responsive design for mobile and desktop
- Animated transitions and smooth interactions
- Accessibility-focused interface
- Multi-language support considerations

### Community Features
- Success stories and testimonials from recovered individuals
- Upcoming events and awareness programs
- Feedback system for user input
- Government partnership information

## Components and Pages

### Core Pages
- **HomePage**: Main landing page with hero section, stats dashboard, and featured facilities
- **CenterDetailPage**: Detailed view of individual facilities with comprehensive information
- **IrcaCenterDetailPage**: Specialized detail page for IRCA centers
- **GovernmentCentersPage**: List of government-run IRCA centers
- **PrivateCentersPage**: List of private IRCA centers
- **GovernmentHospitalsPage**: Government hospitals with psychiatric services
- **PrivateHospitalsPage**: Private hospitals with psychiatric services
- **PsychiatristsPage**: Directory of psychiatrists and mental health professionals
- **EventsPage**: Calendar of upcoming workshops and awareness programs
- **FAQPage**: Frequently asked questions about addiction recovery
- **FeedbackPage**: User feedback and testimonial submission

### Key Components
- **Header**: Navigation with district-based filtering
- **Footer**: Contact information and quick links
- **MapComponent**: Interactive Leaflet-based map
- **ScrollToTopButton**: Navigation aid for long pages
- **UI Components**: Reusable interface elements (buttons, cards, forms, etc.)
